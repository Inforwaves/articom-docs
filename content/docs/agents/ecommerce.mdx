## Overview

The E-commerce Agent is designed to handle customer interactions in online retail environments, providing product recommendations, order support, and shopping assistance.

## Features

### Product Discovery
- Help customers find products based on their needs
- Provide detailed product information
- Compare products and suggest alternatives
- Answer questions about specifications and availability

### Order Management
- Track order status
- Process returns and exchanges
- Update shipping information
- Handle order modifications

### Customer Support
- Answer frequently asked questions
- Resolve common issues
- Escalate complex problems to human agents
- Provide personalized recommendations

## Best Practices

1. **Personalization**: Use customer history to provide relevant recommendations
2. **Quick Responses**: Ensure fast response times for order inquiries
3. **Clear Communication**: Provide transparent information about shipping and returns
4. **Proactive Updates**: Notify customers about order status changes

## Use Cases

- Product recommendations
- Cart abandonment recovery
- Order status inquiries
- Return and refund requests
- Inventory checks
- Promotional campaigns
