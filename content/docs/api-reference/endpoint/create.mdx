# Create Agent

Create a new AI agent with custom configuration and capabilities.

## Endpoint

```
POST /v1/agents
```

## Request

### Headers

```
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

### Body Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | Name of the agent |
| `type` | string | Yes | Agent type (support, sales, etc.) |
| `description` | string | No | Agent description |
| `configuration` | object | No | Agent configuration settings |
| `capabilities` | array | No | List of enabled capabilities |
| `metadata` | object | No | Custom metadata |

### Configuration Object

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `language` | string | "en" | Primary language |
| `personality` | string | "professional" | Agent personality |
| `temperature` | number | 0.7 | Response creativity (0-1) |
| `max_tokens` | number | 1000 | Maximum response length |
| `model` | string | "gpt-4" | AI model to use |

## Example Request

```bash
curl https://api.articom.io/v1/agents \
  -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Customer Support Agent",
    "type": "support",
    "description": "Handles customer inquiries and support tickets",
    "configuration": {
      "language": "en",
      "personality": "helpful",
      "temperature": 0.7,
      "max_tokens": 1500
    },
    "capabilities": [
      "chat",
      "ticket_creation",
      "knowledge_base"
    ],
    "metadata": {
      "department": "support",
      "priority": "high"
    }
  }'
```

## Code Examples

### JavaScript/TypeScript

```javascript
const articom = require('@articom/sdk');

const client = new articom.Client({
  apiKey: 'your_api_key_here'
});

const agent = await client.agents.create({
  name: 'Customer Support Agent',
  type: 'support',
  configuration: {
    language: 'en',
    personality: 'helpful'
  }
});

console.log('Agent created:', agent.id);
```

### Python

```python
import articom

client = articom.Client(api_key='your_api_key_here')

agent = client.agents.create(
    name='Customer Support Agent',
    type='support',
    configuration={
        'language': 'en',
        'personality': 'helpful'
    }
)

print(f'Agent created: {agent.id}')
```

### Node.js (Fetch)

```javascript
const response = await fetch('https://api.articom.io/v1/agents', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key_here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Customer Support Agent',
    type: 'support',
    configuration: {
      language: 'en',
      personality: 'helpful'
    }
  })
});

const agent = await response.json();
```

## Agent Types

- `support` - Customer support agent
- `sales` - Sales and lead qualification
- `ecommerce` - E-commerce assistance
- `restaurant` - Restaurant reservations
- `event` - Event registration
- `chat` - General chat support
- `ticketing` - Ticket management
- `complaints` - Complaint handling

## Capabilities

Available capabilities:
- `chat` - Text-based conversations
- `voice` - Voice interactions
- `ticket_creation` - Create support tickets
- `knowledge_base` - Access knowledge base
- `analytics` - Usage analytics
- `sentiment_analysis` - Detect sentiment
- `multilingual` - Multiple languages
- `escalation` - Escalate to humans

## Rate Limits

- **Free**: 10 agents
- **Pro**: 100 agents
- **Enterprise**: Unlimited
