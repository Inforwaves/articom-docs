---
title: Delete Agent
description: Delete an existing agent and all associated data.
---

## Endpoint

```
DELETE /v1/agents/{agent_id}
```

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_id` | string | Yes | The unique agent identifier |

## Request

### Headers

```
Authorization: Bearer YOUR_API_KEY
```

### Example Request

```bash
curl -X DELETE https://api.articom.io/v1/agents/agent_1a2b3c4d5e6f \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Response

### Success Response (200 OK)

```json
{
  "success": true,
  "data": {
    "id": "agent_1a2b3c4d5e6f",
    "deleted": true,
    "deleted_at": "2025-11-20T10:00:00Z"
  }
}
```

### Error Response (404 Not Found)

```json
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Agent not found"
  }
}
```

## Important Notes

<Callout type="warn">
**Warning**: Deleting an agent is permanent and cannot be undone. All associated data including conversations, analytics, and configurations will be deleted.
</Callout>

### What Gets Deleted

When you delete an agent, the following data is permanently removed:

- Agent configuration and settings
- All conversation history
- Analytics and statistics
- Custom training data
- Webhooks and integrations
- API keys specific to the agent

### Data Retention

- **Immediate**: Agent is deactivated and stops responding
- **7 days**: Soft delete period (contact support for recovery)
- **30 days**: Complete data deletion from backups

## Soft Delete (Deactivate)

If you want to temporarily disable an agent without deleting it, use the update endpoint:

```bash
curl -X PATCH https://api.articom.io/v1/agents/agent_1a2b3c4d5e6f \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}'
```

## Bulk Delete

Delete multiple agents at once:

### Endpoint

```
POST /v1/agents/bulk-delete
```

### Request Body

```json
{
  "agent_ids": [
    "agent_1a2b3c4d5e6f",
    "agent_2b3c4d5e6f7g",
    "agent_3c4d5e6f7g8h"
  ]
}
```

### Example Request

```bash
curl -X POST https://api.articom.io/v1/agents/bulk-delete \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_ids": [
      "agent_1a2b3c4d5e6f",
      "agent_2b3c4d5e6f7g"
    ]
  }'
```

### Response

```json
{
  "success": true,
  "data": {
    "deleted_count": 2,
    "failed_count": 0,
    "deleted_agents": [
      {
        "id": "agent_1a2b3c4d5e6f",
        "status": "deleted"
      },
      {
        "id": "agent_2b3c4d5e6f7g",
        "status": "deleted"
      }
    ]
  }
}
```

## Code Examples

### JavaScript/TypeScript

```javascript
const articom = require('@articom/sdk');

const client = new articom.Client({
  apiKey: 'your_api_key_here'
});

// Delete single agent
await client.agents.delete('agent_1a2b3c4d5e6f');
console.log('Agent deleted');

// Bulk delete
await client.agents.bulkDelete([
  'agent_1a2b3c4d5e6f',
  'agent_2b3c4d5e6f7g'
]);
```

### Python

```python
import articom

client = articom.Client(api_key='your_api_key_here')

# Delete single agent
client.agents.delete('agent_1a2b3c4d5e6f')
print('Agent deleted')

# Bulk delete
client.agents.bulk_delete([
    'agent_1a2b3c4d5e6f',
    'agent_2b3c4d5e6f7g'
])
```

### Node.js (Fetch)

```javascript
// Delete single agent
const response = await fetch(
  'https://api.articom.io/v1/agents/agent_1a2b3c4d5e6f',
  {
    method: 'DELETE',
    headers: {
      'Authorization': 'Bearer your_api_key_here'
    }
  }
);

const result = await response.json();
console.log('Deleted:', result.data.deleted);
```

## Before Deleting

Consider these alternatives before permanently deleting an agent:

1. **Deactivate**: Set status to inactive
2. **Archive**: Move to archive for later reference
3. **Export Data**: Download conversation history and analytics
4. **Transfer**: Move ownership to another team member

## Export Data Before Deletion

```bash
# Export agent data
curl https://api.articom.io/v1/agents/agent_1a2b3c4d5e6f/export \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -o agent_data.json
```

## Error Responses

### 403 Forbidden

```json
{
  "success": false,
  "error": {
    "code": "forbidden",
    "message": "You don't have permission to delete this agent"
  }
}
```

### 409 Conflict

```json
{
  "success": false,
  "error": {
    "code": "conflict",
    "message": "Cannot delete agent with active conversations"
  }
}
```
