# Get Agent

Retrieve information about a specific agent or list all agents in your account.

## Get Single Agent

Retrieve details of a specific agent by ID.

### Endpoint

```
GET /v1/agents/{agent_id}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_id` | string | Yes | The unique agent identifier |

### Example Request

```bash
curl https://api.articom.io/v1/agents/agent_1a2b3c4d5e6f \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response

```json
{
  "success": true,
  "data": {
    "id": "agent_1a2b3c4d5e6f",
    "name": "Customer Support Agent",
    "type": "support",
    "status": "active",
    "configuration": {
      "language": "en",
      "personality": "helpful",
      "temperature": 0.7
    },
    "capabilities": ["chat", "ticket_creation"],
    "created_at": "2025-11-20T10:00:00Z",
    "updated_at": "2025-11-20T10:00:00Z",
    "stats": {
      "total_conversations": 1543,
      "avg_response_time": 1.2,
      "satisfaction_rate": 4.7
    }
  }
}
```

## List Agents

Retrieve a list of all agents in your account.

### Endpoint

```
GET /v1/agents
```

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | number | 10 | Number of results (1-100) |
| `offset` | number | 0 | Pagination offset |
| `type` | string | - | Filter by agent type |
| `status` | string | - | Filter by status (active/inactive) |
| `sort` | string | created_at | Sort field |
| `order` | string | desc | Sort order (asc/desc) |

### Example Request

```bash
curl "https://api.articom.io/v1/agents?limit=20&type=support" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response

```json
{
  "success": true,
  "data": [
    {
      "id": "agent_1a2b3c4d5e6f",
      "name": "Customer Support Agent",

      "id": "agent_2b3c4d5e6f7g",
      "name": "Sales Assistant",
      "type": "sales",
```

## Get Agent Statistics

Retrieve detailed statistics for an agent.

### Endpoint

```
GET /v1/agents/{agent_id}/stats
```

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `period` | string | Time period (day/week/month) |
| `start_date` | string | Start date (ISO 8601) |
| `end_date` | string | End date (ISO 8601) |

### Example Request

```bash
curl "https://api.articom.io/v1/agents/agent_1a2b3c4d5e6f/stats?period=week" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response

```json
{
  "success": true,
  "data": {
    "agent_id": "agent_1a2b3c4d5e6f",
    "period": "week",

    },
```

## Code Examples

### JavaScript/TypeScript

```javascript
const articom = require('@articom/sdk');

const client = new articom.Client({
  apiKey: 'your_api_key_here'
});

// Get single agent
const agent = await client.agents.get('agent_1a2b3c4d5e6f');
console.log(agent.name);

// List agents
const agents = await client.agents.list({
  limit: 20,
  type: 'support'
});

// Get agent stats
const stats = await client.agents.getStats('agent_1a2b3c4d5e6f', {
  period: 'week'
});
```

### Python

```python
import articom

client = articom.Client(api_key='your_api_key_here')

# Get single agent
agent = client.agents.get('agent_1a2b3c4d5e6f')
print(agent.name)

# List agents
agents = client.agents.list(limit=20, type='support')

# Get agent stats
stats = client.agents.get_stats('agent_1a2b3c4d5e6f', period='week')
```

### cURL

```bash
# Get single agent
curl https://api.articom.io/v1/agents/agent_1a2b3c4d5e6f \
  -H "Authorization: Bearer YOUR_API_KEY"

# List agents with filters
curl "https://api.articom.io/v1/agents?limit=20&type=support&status=active" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Get agent statistics
curl "https://api.articom.io/v1/agents/agent_1a2b3c4d5e6f/stats?period=month" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Error Responses

```json
{
  "success": false,
  "error": {
    "code": "not_found",
    "message": "Agent not found"
  }
}
```

```json
{
  "success": false,
  "error": {
    "code": "invalid_parameters",
    "message": "Invalid query parameters",
    "details": {
      "limit": "Must be between 1 and 100"
    }
  }
}
```
