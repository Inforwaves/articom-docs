## Introduction

Cursor is an AI-first code editor that reimagines how developers write code, combining the power of AI with a familiar editing experience.

## Features

### AI Pair Programming
- Real-time code suggestions
- Context-aware completions
- Multi-file editing support
- Natural language commands

### Smart Editing
- CMD+K for inline editing
- Chat with your codebase
- Reference specific files or docs
- Batch code modifications

### Codebase Understanding
- Ask questions about your code
- Navigate complex projects
- Find implementation patterns
- Understand dependencies

## Getting Started

```bash
# Download Cursor from cursor.sh
# Import your VS Code settings
# Start coding with AI assistance
```

## Key Features

### CMD+K (Quick Edit)
Generate or modify code inline with natural language:

```
CMD+K: "Add error handling to this function"
CMD+K: "Convert this to TypeScript"
CMD+K: "Add JSDoc comments"
```

### Chat Interface
Ask questions about your codebase:

```
"How does authentication work in this app?"
"Where is the user model defined?"
"Show me all API endpoints"
```

### Composer Mode
Make changes across multiple files simultaneously:

```
"Add a new feature for user notifications across the app"
"Refactor the database layer to use Prisma"
```

## Best Practices

1. **Specific Requests**: Be clear and specific in your prompts
2. **Context**: Reference relevant files with @mentions
3. **Iterative**: Build features incrementally
4. **Review**: Always review AI suggestions
5. **Learn**: Use AI explanations to understand patterns

## Keyboard Shortcuts

- `CMD+K` - Inline edit
- `CMD+L` - Open chat
- `CMD+I` - Ask about selection
- `CMD+Shift+K` - Composer mode
