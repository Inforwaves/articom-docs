## Custom API Integration

For custom integrations, Articom provides a comprehensive REST API:

```json
{
  "conversation_id": "conv_123",
  "customer": {
    "name": "John Doe",
    "email": "john@example.com"
  },
  "channel": "whatsapp",
  "messages": [
    {
      "role": "user",
      "content": "What are your business hours?"
    },
    {
      "role": "assistant",
      "content": "We're open Monday-Friday 9 AM to 6 PM."
    }
  ]
}
```
