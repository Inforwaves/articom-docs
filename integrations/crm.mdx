## Configuration Options

<AccordionGroup>
  <Accordion title="Field Mapping">
    Map conversation data to CRM fields:
    - Customer name → Contact Name
    - Email → Contact Email
    - Phone → Contact Phone
    - Conversation topic → Case Subject
    - Sentiment → Custom Sentiment Field
  </Accordion>
  
  <Accordion title="Sync Rules">
    Define when and what to sync:
    - Create contact on first conversation
    - Update contact on subsequent chats
    - Create case for unresolved issues
    - Log all conversations as activities
  </Accordion>
  
  <Accordion title="Automation Triggers">
    Set up automated actions:
    - Send email when deal closes
    - Assign to sales rep based on topic
    - Create task for follow-up
    - Update opportunity stage
  </Accordion>
</AccordionGroup>

## Best Practices

<Note>
  **Tip**: Start with basic contact sync, then gradually add more complex automations as you get comfortable with the integration.
</Note>

1. **Clean CRM Data**: Ensure CRM data is clean before integration
2. **Test Thoroughly**: Use sandbox/test environment first
3. **Monitor Sync**: Regularly check sync logs for errors
4. **User Training**: Train team on integrated workflows
5. **Regular Reviews**: Monthly review of automation rules
